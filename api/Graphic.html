Class Graphic

Graphic is a simple drawing api that allows for basic drawing operations.

Constructor

Graphic ( )

Configuration Attributes

autoSize - Boolean

Determines how the size of instance is calculated. If true, the width and height are determined by the size of the contents. If false, the width and height values are either explicitly set or determined by the size of the parent node's dimensions.
Default Value: false

contentBounds - Object

Object containing size and coordinate data for the content of a Graphic in relation to the coordSpace node.

id - String

Unique id for class instance.

node - HTMLElement

The html element that represents to coordinate system of the Graphic instance.

pointerEvents - String

Indicates the pointer-events setting for the svg:svg element.

resizeDown - Boolean

When overflow is set to true, by default, the viewBox will resize to greater values but not values. (for performance) When resizing the viewBox down is desirable, set the resizeDown value to true.

shapes - Object

Key value pairs in which a shape instance is associated with its id.

x - Number

Indicates the x-coordinate for the instance.

y - Number

Indicates the y-coordinate for the instance.

Draws a wedge.

yRadius <Number> [optional] y radius for wedge.
