Class ToolSet - extends Component

A base class for ToolSet, providing:
  • Widget Lifecycle (initializer, renderUI, bindUI, syncUI, destructor)
  • Managed collection for ToolItems
Quick Example:
var instance = new A.ToolSet({
tools: [
{
id: 'close',
icon: 'close'
},
{
id: 'expand',
icon: 'plus'
}
]
}).render();
